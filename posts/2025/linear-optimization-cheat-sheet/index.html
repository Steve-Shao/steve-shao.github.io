Core Concepts

  • Decision Variables: $x_1, \dots, x_n$
  • Objective Function: $\max \ c^\top x$
  • Constraints:
    • Inequalities: $Ax \leq b$ or $Ax \geq b$.
    • Equalities: $Ax = b$.
  • Feasible Region: convex polytope defined by constraints.
  • Extreme Points: polytope vertices.
  • Optimal Solution: vertex of feasible region that optimizes objective.

Standard vs. Canonical Forms, Duality

  • Standard Form:
    $\max c^\top x$ s.t. $Ax = b$, $x \geq 0$
    Dual:
    $\min b^\top y$ s.t. $A^\top y \geq c$
  • Canonical Form:
    $\max c^\top x$ s.t. $Ax \leq b$, $x \geq 0$
    Dual:
    $\min b^\top y$ s.t. $A^\top y \geq c$, $y \geq 0$
  • Standard to Canonical:
    Add slack variables to inequalities.
  • Canonical to Standard:
    $Ax = b$ to $Ax \leq b$, $-Ax \leq -b$.
  • Duality Notes
    Inequality → non-negative variable
    Equality → unrestricted variable
    Dual of dual is primal

Key Theorems

  1. Fundamental Theorem of LP:
    An optimal solution (if exists) lies at a vertex of the feasible region.
  2. Weak Duality (for primal max):
    $obj_{\mathrm{dual}} \geq obj_{\mathrm{primal}}$
  3. Strong Duality:
    $obj_{\mathrm{dual}} = obj_{\mathrm{primal}}$
  4. Complementary Slackness:
    $x_i \geq 0 \implies y_j = 0$
    for active constraints.

Simplex Method:

  1. Use Standard Form
    $\max c^\top x$ s.t. $Ax = b$, $x \geq 0$

  2. Basis Setup
    Partition:

    • $A = [B | N]$,
    • $x = [x_B | x_N]$,
    • $c = [c_B | c_N]$.

    Basic solution:

    • $x_B = B^{-1}b$,
    • $x_N = 0$.
    • Feasible if $x_B \geq 0$.

  3. Objective in Non-Basic Variables
    $z = c_B^\top B^{-1}b $
    $+ (c_N^\top - c_B^\top B^{-1}N)x_N$.
    Reduced costs:
    $\bar{c}_N = c_N - N^\top(B^{-1})^\top c_B$.

  4. Optimality Condition
    If $\bar{c}_N \leq 0$,
    current solution is optimal.
    Else, select entering variable
    $x_j$ with $\bar{c}_j > 0$.

  5. Pivot Selection

    • Compute direction $d = B^{-1}A_j$ (column of entering variable).
    • If $d \leq 0$, problem is unbounded.
    • Min ratio test: $\theta = \min((B^{-1}b)_i / d_i | d_i > 0)$.
    • Update basis: Replace leaving variable (row with min ratio) with $x_j$.

  6. Repeat until optimal/unbounded.

Key Equations:

  • $x_B = B^{-1}b - B^{-1}Nx_N$
  • Reduced costs: $\bar{c}_N = c_N - N^\top(B^{-1})^\top c_B$
  • Direction: $d = B^{-1}A_j$

Logical to Linear via Big-M

  1. Implication:
    if binary $x = 1$ then $a^Ty \geq b$:
    $a^Ty \geq b - M(1 - x)$
  2. Conditional:
    If $x > 0$ then $a^Ty \geq b$:
    $a^Ty \geq b - M(1 - z)$,
    $x > - Mz$, $z$ binary
  3. Either-Or:
    either $A_1y \leq b_1$ or $A_2y \leq b_2$:
    $A_1y \leq b_1 + Mz$,
    $A_2y \leq b_2 + M(1 - z)$, $z$ binary
  4. Fixed Cost:
    If $y > 0$ then cost $= cy + f$:
    $y \leq Mz$, $z$ binary,
    add $fz$ to objective

Network Flow Problems
Automatically yields integer solutions.

Transportation Problem

  • Formulation:
    $\min \sum_{i,j} c_{ij}x_{ij}$
    s.t. $\sum_j x_{ij} = s_i$ (supply),
    $\sum_i x_{ij} = d_j$ (demand),
    $x_{ij} \geq 0$.
  • Structure: Bipartite graph; solved via transportation simplex.

Transshipment Problem

  • Formulation:
    $\min \sum_{(i,j)} c_{ij}x_{ij}$
    s.t. $\sum_{j} x_{ij} - \sum_{j} x_{ji} = b_i$
    (flow balance),
    $0 \leq x_{ij} \leq u_{ij}$.
  • Structure: Nodes can be suppliers, demanders, or transshipment points.

Maximum Flow

  • Formulation:
    $\max; v$
    s.t. $\sum_{j} x_{ij} - \sum_{j} x_{ji} = $
    $v \text{ or } -v \text{ or } 0$,
    $0 \leq x_{ij} \leq u_{ij}$.
  • Key Result: Max-Flow Min-Cut.

Shortest Path

  • Formulation:
    $\min \sum_{(i,j)} c_{ij}x_{ij}$
    s.t. $\sum_{j} x_{ij} - \sum_{j} x_{ji} = $
    $1 \text{ or } -1 \text{ or } 0$,
    $x_{ij} \geq 0$.
  • Special Case: Dijkstra’s algorithm (non-negative costs).

Minimum Cost Flow

  • General Formulation:
    $\min \sum_{(i,j)} c_{ij}x_{ij}$
    s.t. $\sum_{j} x_{ij} - \sum_{j} x_{ji} = b_i$
    (flow balance),
    $l_{ij} \leq x_{ij} \leq u_{ij}$.
  • Unifies: Transportation, transshipment, shortest path, max flow.

Assignment Problem

  • Formulation:
    $\min \sum_{i,j} c_{ij}x_{ij}$
    s.t. $\sum_j x_{ij} = 1$, $\sum_i x_{ij} = 1$,
    $x_{ij} \in {0,1}$.
  • Key Insight: Solved via LP relaxation (Hungarian algorithm).
